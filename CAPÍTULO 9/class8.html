Trabalhando com Listas

Ordered Lists:

Listas Ordenadas, como o nome já diz são listas nas quais a ordem dos itens faz diferença.
Veja o exemplo a seguir:

    Cronograma Diário:
  1. Acordar
  2. Tomar Café
  3. Ajudar a Vó
  4. Estudar Musica
    1. Teoria Musical
    2. Prática da Guitarra
  5. Almoçar
    1. Lavar a Louça
    2. Escovar os Dentes
  6. Estudar Programação
    1. HTML / CSS:
      Segunda, Terça, Quinta
    2. JavaScript:
      Quarta, Sexta, Sábado
  7. Comer algo
  8. Ajudar a Vó
    1. Guardar a Louça
    2. Levar o Lixo:
      Segunda, Quarta, Sexta
    3. Comprar Pão
  9. Estudar Repertório na Guitarra

Unordered Lists:

As Listas não ordenadas, como o nome sugere NÃO númeram de forma ordenada os itens.
Confira o exemplo:

    Algumas das Minhas Bandas Favoritas:
  • Led Zeppelin
  • Radiohead
  • Soundgarden
  • Beatles
  • Green Day
  • The Doors

Definition List

Listas de Definição são como Dicionários, temos as tags filhas <dt> (seria o termo em questão) e <dd> (seria a sua definição).
Veja este exemplo:

HTML
É uma Linguagem de Marcação para criação de sites.
CSS
Linguagem de Marcação para a estilização de sites.
JavaScript
Linguagem de Programação usada para dar funcionalidade e interatividade à sites.
